enum TplEventTypeMask

typedef enum {
  TPL_EVENT_MASK_TEXT         = 1 << 0,
  TPL_EVENT_MASK_CALL         = 1 << 1,
  TPL_EVENT_MASK_ANY          = 0xffff
} TplEventTypeMask;

Mask used to filter type of TplEvent returned.

 Mask to TplTextEvent

TPL_EVENT_MASK_CALL

 Mask to TplCallEvent

TPL_EVENT_MASK_ANY
